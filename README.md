# Exc1PersonalReg
Övningsuppgift ett Fullstack kurs

Använder klasserna Person och Register för att skapa personaluppgifter och registrera dem.
Person är ganska självgående och kan såklart utökas med fler egenskaper/attribut.

Tror mig lyckats uppfylla kraven:
personalregister. De har endast två krav:
1. Registret skall kunna ta emot och lagra anställda med namn och lön. (via inmatning
i konsolen, inget krav på persistent lagring)
1. Programmet skall kunna skriva ut registret i en konsol.

Lösningen finns på master branch och bygger bra i Rider IDE.
